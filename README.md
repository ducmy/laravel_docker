
###### Choose your cluster

```shell
# .env
APP_SRC=<your_cluster>
```
###### Choose your cluster
`$ mkdir -p projects/<your_cluster>`
`$ cd projects/<your_cluster> && mkdir source`

###### Start docker
Go to root dir
`$ docker-composer up -d --build`
